# Smart Agro Protection System for Preventing Crop Damage by Animals using Ultrasonic Alarm

## Team Name
Green Innovators

## Problem Statement
Farmers often face significant crop losses due to wild animals entering agricultural fields. Traditional protection methods are costly, require constant monitoring, and may harm animals. An efficient, affordable, and eco-friendly solution is needed to protect crops.

## Solution
The Smart Agro Protection System detects animal movement near crop fields using sensors and activates an ultrasonic alarm. The high-frequency sound helps deter animals without causing physical harm, reducing crop damage and improving agricultural productivity.

## Objectives
- Prevent crop damage caused by animals.
- Provide a low-cost and eco-friendly solution.
- Reduce farmers' losses.
- Minimize human intervention.
- Improve agricultural sustainability.

## Components Used
- Arduino Uno
- PIR Motion Sensor
- Ultrasonic Sensor (HC-SR04)
- Active Buzzer
- LED
- Resistors
- Breadboard
- Jumper Wires
- Power Supply/Battery

## Working Principle
1. PIR sensor detects movement.
2. Arduino processes sensor data.
3. Ultrasonic alarm and buzzer are activated.
4. Animals move away from the protected area.
5. System returns to monitoring mode.
